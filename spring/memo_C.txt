・TについてはGenericsを参照・・・設計図クラスに下記のような記述（public class PairMap<T>）をすることで、実行クラスでインスタンスを定義してインスタンス毎に変数の中身を変えることができる。
PairMap pair = new PairMap<String>("ひとつめ", "ふたつめ");
PairMap pair2 = new PairMap<Integer>(4, 2);

public class PairMap<T> {

  private T first;
  private T second;

  public T first() {
    return this.first;
  }

  public T second() {
    return this.second;
  }

  public PairMap(T first, T second) {
    this.first = first;
    this.second = second;
  }
}
